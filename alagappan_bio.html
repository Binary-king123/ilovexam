iLoveExams – Web Mock Test Platform
A focused exam-practice platform built for NEET, JEE, CUET and SAT, designed to look and feel like the real exam interface but without ads and distractions.
Role: Founder · Full-stack builder
With iLoveExams, I wanted to solve a simple problem: students like me needed a place to open and start a mock test instantly. No clutter, no confusion — just exam-style questions, timers, marking for review, and clear analysis after each test.
What it offers
Mock tests, exam-style UI, question review, instant scoring, and performance analytics.
Who it helps
NEET, JEE, CUET, SAT aspirants who want realistic practice from home, hostel or college.
Tech focus
Web application design, question handling, timing logic, result & analytics flow.

Overlay App – On-screen helper for meetings & classes
A Windows overlay application that sits on top of other windows and gives quick controls for paste, screenshots and auto-typing — very handy during online meetings and day-to-day work.
Role: Desktop developer · C++
During calls, online classes, coding sessions and interviews, switching between tabs and windows wastes time. This overlay app floats on top and provides simple buttons and hotkeys, so I can quickly paste text, take screenshots, or auto-type content from the clipboard — all without breaking focus.
Key idea
Make common actions — paste, screenshot, typing — available from a small control panel while other apps (like meeting tools, chats or IDEs) stay in front.
Useful for
Online meetings, live classes, note-taking, answering questions, and demo sessions where speed matters.
Technical notes
Windows desktop app with overlay window, WebView2 integration, hotkeys, keyboard hook and clipboard handling.

Object Detection using Machine Learning
A project that uses ML-based models to detect objects from images or video and mark them on the screen.
Role: ML learner · Implementation
This project helped me understand how computer vision can “see” things — detecting objects, drawing boxes around them and updating frames in real time. It’s a strong base for future work in security, automation and smart cameras.
Focus
ML pipeline, model inference, feeding image/video data and visualizing detection results.
Where it’s useful
CCTV monitoring, smart attendance, traffic analysis, or any place where the system should know “what is in the frame”.
What I learned
Basics of object detection models, working with frames, and how ML connects to real-world camera input.

Decentralized Farmer–Consumer Transaction Platform (Solana)
A blockchain concept where farmers and consumers interact directly, using the Solana network for secure and transparent transactions.
Role: Blockchain learner · Concept build
With this project, I explored how blockchain can support real-world use-cases. The idea is simple: farmers list their produce, consumers connect directly, and Solana smart contracts help handle secure transactions and records.
Goal
Reduce middlemen and increase transparency in farmer–consumer interactions using decentralized tech.
Tech
Solana blockchain concepts, smart contract logic, basic frontend to explore the flow.
Takeaway
Understanding how on-chain transactions work and how to design flows where trust comes from the system, not a single person.
