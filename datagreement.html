Data Processing Agreement (DPA)

This Data Agreement outlines how I Love Exams processes and protects user data in compliance with global standards.

1. Data Ownership

All user data, including exam responses and analytics, remains the property of the user or the educational institution.

2. Data Storage

Data is encrypted and stored in secure servers with daily backups.

3. Data Access & Sharing

We do not share your data with third parties unless required by law.

4. Data Deletion

Users can request data deletion by contacting support@ilovexams.com. Requests are fulfilled within 30 days.
